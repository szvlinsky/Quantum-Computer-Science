# Zbiór wzorów fizycznych

---

## 1. Wzory – Kinematyka

| Nazwa | Wzór | Jednostka | Opis |
|-------|------|------------|------|
| Funkcja położenia | x(t) = x₀ + V₀·t + (1/2)·a·t² | m | x₀ – położenie początkowe; V₀ – prędkość początkowa; a – przyspieszenie |
| Funkcja prędkości | V(t) = V₀ + a·t | m/s | pierwsza pochodna funkcji położenia; ruch prostoliniowy |
| Funkcja przyspieszenia | a(t) = const | m/s² | druga pochodna funkcji położenia; ruch jednostajnie przyspieszony |
| Droga w ruchu jednostajnym | s = v·t | m | v – prędkość stała; t – czas ruchu |
| Droga w ruchu jednostajnie przyspieszonym | s = V₀·t + (1/2)·a·t² | m | suma drogi wynikającej z prędkości początkowej i przyspieszenia |
| Prędkość średnia | vₛᵣ = s / t | m/s | stosunek całkowitej drogi do czasu ruchu |
| Przemieszczenie | Δx = x₂ - x₁ | m | zmiana położenia punktu w czasie ruchu |
| Związek między prędkością a drogą | v² = V₀² + 2·a·s | m²/s² | równanie bez czasu; ruch jednostajnie przyspieszony |
| Rzut pionowy – wysokość maks. | hₘₐₓ = V₀² / (2·g) | m | g – przyspieszenie ziemskie (~9.81 m/s²) |
| Czas wznoszenia | t_w = V₀ / g | s | czas od startu do zatrzymania się w górze |
| Czas spadku | tₛ = √(2·h / g) | s | czas swobodnego spadania z wysokości h |
| Droga w ruchu jednostajnie opóźnionym | s = V₀·t - (1/2)·a·t² | m | a – opóźnienie (ujemne przyspieszenie) |
| Ruch po okręgu – prędkość kątowa | ω = Δφ / Δt | rad/s | ω – prędkość kątowa; φ – kąt obrotu |
| Ruch po okręgu – prędkość liniowa | v = ω·r | m/s | v – prędkość liniowa; r – promień okręgu |
| Przyspieszenie dośrodkowe | a_d = v² / r | m/s² | kierunek przyspieszenia – do środka okręgu |

## 2. Wzory – Dynamika

| Nazwa | Wzór | Jednostka | Opis |
|-------|------|------------|------|
| Druga zasada dynamiki Newtona | F = m·a | N (kg·m/s²) | F – siła wypadkowa; m – masa; a – przyspieszenie |
| Pierwsza zasada dynamiki Newtona | ΣF = 0 → v = const | – | brak sił lub siły się równoważą – ruch jednostajny |
| Trzecia zasada dynamiki Newtona | F₁₂ = -F₂₁ | N | akcja i reakcja mają tę samą wartość, przeciwny zwrot |
| Siła ciężkości | Fg = m·g | N | g ≈ 9.81 m/s² – przyspieszenie ziemskie |
| Ciężar ciała | Q = m·g | N | siła, z jaką Ziemia przyciąga ciało |
| Siła sprężystości | Fs = -k·x | N | k – współczynnik sprężystości; x – odkształcenie sprężyny |
| Siła tarcia | Ft = μ·N | N | μ – współczynnik tarcia; N – siła nacisku |
| Siła oporu ruchu | Fop = k·v² lub Fop = b·v | N | zależność siły oporu od prędkości (dla cieczy/gazów) |
| Siła dośrodkowa | F = m·v² / r | N | siła powodująca ruch po okręgu |
| Związek między siłą a pędem | F = Δp / Δt | N | p – pęd ciała; Δp – zmiana pędu w czasie Δt |
| Pęd ciała | p = m·v | kg·m/s | iloczyn masy i prędkości; wektorowa wielkość ruchu |
| Zasada zachowania pędu | Σp_przed = Σp_po | kg·m/s | pęd układu izolowanego jest stały |
| Moment siły | M = F·r | N·m | M – moment względem punktu; r – ramię siły |
| Warunek równowagi sił | ΣF = 0 i ΣM = 0 | – | układ jest w stanie równowagi statycznej |
| Siła nacisku | N = Fg·cos(α) | N | N – składowa siły ciężkości prostopadła do powierzchni |
